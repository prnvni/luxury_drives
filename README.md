# Luxury Drives 
A full-stack web application that simulates a modern luxury car rental platform. Users can browse vehicles by category, view detailed specifications, and place bookings through a dynamic, database-driven interface. The system also includes administrative functionality for managing vehicles and bookings.

## Features
-Category-based vehicle listings (Luxury, Sports, SUVs, Sedans)
-Dynamic booking system with database integration
-User authentication and session management
-Admin dashboard for managing cars and bookings
-Responsive and interactive frontend UI
-Backend architecture using Java Servlets

## Tech Stack
-HTML
-CSS
-JavaScript
-Java Servlets (JSP/Servlet-based architecture)
-MySQL
